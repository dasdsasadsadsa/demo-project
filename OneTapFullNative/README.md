# OneTap Full Native

Kivy/Buildozer 대신 순수 Android Java로 만든 초소형 WebView 브라우저 MVP입니다.

## 기능
- 상단 URL 입력
- 안드로이드 시스템 WebView
- 이동 버튼
- 전체화면 버튼
- 전체화면 진입 시 상태표시줄/내비게이션바/앱 UI 숨김
- 뒤로가기 버튼으로 전체화면 복귀

## 빌드

Android Studio 또는 AndroidIDE에서 이 폴더를 열고 빌드하세요.

명령어 환경에서는:

```bash
gradle :app:assembleDebug
```

APK 위치:

```text
app/build/outputs/apk/debug/app-debug.apk
```
